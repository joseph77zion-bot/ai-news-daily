# AI ビジネス デイリーブリーフィング

毎朝8時に世界のビジネスに役立つAI最新情報をまとめたインタラクティブなダッシュボードです。

## 特徴

- 📊 **6つのカテゴリ分類**: 投資・市場動向、AIモデル・技術、労働・人材、セキュリティ・規制、地域・産業別
- 📈 **インタラクティブなチャート**: Chart.jsを使用した動的なデータ可視化
- 🌍 **世界中の主要メディアから情報収集**: Reuters、Bloomberg、Forbes、KPMG、EYなど
- 🔄 **毎日自動更新**: 毎朝8時（JST）に最新情報を反映

## コンテンツ

- **KPIカード**: 主要な数値指標を一目で把握
- **ダッシュボード**: 複数のチャートでトレンドを可視化
- **Key Takeaways**: 重要なポイントをテーブルで整理
- **詳細記事**: 各カテゴリの深掘り情報

## 利点

1. **データをより直感的に探索できる** - インタラクティブなチャートで複雑なデータを視覚化
2. **トレンドをよりよく理解できる** - グラフで数値の変化と方向性を一目で把握
3. **簡単に保存または共有できる** - 静的HTMLファイルでURLを共有するだけで即座に共有可能
